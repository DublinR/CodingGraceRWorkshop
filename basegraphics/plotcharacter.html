\section*{Plot Character} The default plot character is a “hollow” circle. We can change the type of plot character used by specifying a value to the \textbf{\texttt{pch=}} argument. We can change the colour using the \textbf{\texttt{col=}} argument. \begin{framed} \begin{verbatim} plot(x,y, pch = 16, col="red") \end{verbatim} \end{framed} \section*{Changing your plot character} For a list of plot characters see: \begin{verbatim} http://voteview.com/symbols_pch.htm \end{verbatim} \end{document}
